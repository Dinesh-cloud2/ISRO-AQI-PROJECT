# 🌍 ISRO AQI Intelligence Platform

An AI-powered Air Quality Intelligence Platform that combines satellite data, weather information, and machine learning to predict Air Quality Index (AQI).

## ✨ Features

- 🛰️ Google Earth Engine Integration
- 🌫️ AQI Prediction using Machine Learning
- 📊 Interactive Dashboard
- 📈 Analytics Charts
- 🗺️ Live AQI Map
- 🧠 Explainable AI (XAI)
- 🎮 Scenario Simulator
- 📄 PDF Report Generator

- Python
- Streamlit
- Google Earth Engine
- Scikit-learn
- Pandas
- Plotly
- Folium
- SHAP
